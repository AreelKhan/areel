A real-time visualization tool that lets you peer inside deep learning models as they train. Watch layer activations light up, track gradient flow, and spot vanishing gradients before they kill your training run.

## Features

- Real-time layer activation heatmaps
- Gradient flow visualization across the full network
- Interactive architecture diagrams
- Training metric dashboards with loss curves and learning rate schedules
