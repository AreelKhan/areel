A high-performance distributed task queue built from the ground up in Rust. Designed to handle millions of tasks per second with guaranteed at-least-once delivery and automatic retries.

## Features

- Lock-free work-stealing scheduler
- Redis-backed persistent storage with WAL
- gRPC-based inter-node communication
- Automatic task retry with exponential backoff
